현재 IDE에서 열려 있는 파일(또는 인자로 전달된 파일 경로 `$ARGUMENTS`)의 MD 파일에 아래 세 섹션을 추가해줘.

CLAUDE.md의 형식 규칙을 그대로 따른다:
- 섹션 번호는 기존 섹션 이어서 매긴다
- 섹션 제목: `## 숫자. 제목` 형식
- 코드 예시 반드시 포함 (언어 명시)
- 비교는 주석 또는 표로 시각화
- 마지막 섹션 끝에 한 줄 핵심 요약 포함
- 말투는 `~다` 체

추가할 섹션:
1. **실전 활용 예시** — 실제 상황에서 어떻게 쓰이는지 코드와 함께 2~3가지 케이스로 설명
2. **비표준/기존 방식과의 비교** — 예전 방식 또는 대안과 비교해 왜 이 방식이 나은지 설명
3. **주의사항 또는 한계** — 오해하기 쉬운 점, 쓰면 안 되는 경우, 보안/성능 관련 주의점

파일 경로가 인자로 주어졌으면 해당 파일을 수정하고, 없으면 현재 IDE에서 열려 있는 파일을 수정한다.
